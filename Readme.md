## 📌 Project Overview

SkillSphere is a high-performance digital learning platform built for aspiring developers and technical professionals. The platform combines a clean, futuristic design with powerful functionality to deliver a seamless online learning experience.

From browsing trending courses to managing user profiles and authentication, SkillSphere provides everything needed for a professional educational platform.

---

## ✨ Key Features

### 🔐 Authentication System
- Secure Sign Up and Login
- Better Auth integration
- Real-time form validation
- Password visibility toggle
- Toast notifications for success/error feedback

### 📚 Course Management
- Trending courses section
- Complete course catalog
- Dynamic course detail pages
- Accordion-based syllabus view
- Search functionality for all courses

### 👨‍🎓 Student Dashboard
- Personalized profile page
- Update name and avatar
- Display user role and verification status

### 🎨 Modern UI/UX
- Fully responsive design
- Tailwind CSS 4 + DaisyUI
- Framer Motion animations
- Cyber-inspired gradient styling

### 🛡️ Error Handling
- Custom 404 page
- Route fallback handling
